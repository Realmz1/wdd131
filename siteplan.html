Site Purpose

The purpose of this site is to provide homeowners, businesses, and municipalities with expert tree removal, trimming, and maintenance services.

The site will showcase our services, provide tree care tips, offer free consultation requests, and highlight customer testimonials.

Scenarios

  • How do I know if my tree needs to be removed or just trimmed?
  • What is the cost of tree removal services?
  • How do I schedule a consultation for tree trimming?

Color Schema

The website will use the following colors:

  • Dark Green (#2C5F2D) - Used for headings, logo, and primary call-to-action buttons.
  • Earthy Brown (#8B5A2B) - Used for accents and backgrounds.
  • Soft Beige (#F4E1C6) - Used for the main background color for readability.

Typography

The following fonts will be used:

  • Poppins (sans-serif) - Used for headings and subheadings for a clean, modern look.
  • Lora (serif) - Used for body text to improve readability and add a professional feel.
